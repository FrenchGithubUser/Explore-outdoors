Adding a new spot to the map

You can add a spot that is not referenced yet on the map by different ways :

Option 1 (fast & easy) :

The fast and easy way is by using this form. You won't need any account to add it.
It sends your input into a google spreasheet which I use to update the OSM database everyday.

Option 2 :

The website is relying on the OSM database of markers with the tag "cliff_diving", so you can contribute directly on OSM and your spot will appear as soon as I will have pulled the new spots from their database (the map is showing the spots from a file that is generated once a day from the OSM database).
